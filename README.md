# Customer Behavior & Retention Analysis (SQL)

## Project Overview
This project analyzes customer purchasing behavior using SQL to understand retention patterns, identify repeat customers, and detect customers at risk of churn in an e-commerce setting.

## Business Objectives
- Analyze customer purchasing behavior
- Measure repeat purchase patterns
- Identify high-value customers
- Detect customers at risk of churn

## Dataset
Brazilian E-Commerce Public Dataset (Olist)

Tables used:
- customers
- orders
- order_items

## Tools Used
- PostgreSQL
